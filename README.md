# Quik Static Pages

## Setup

### Required


```
node >= 16 LTS
```

### Install Gulp
```
cd tools
```

```
npm install --global gulp
```
```
npm install --global gulp-cli
```


### Install Yarn

```
npm install --global yarn
```

```
yarn
```

### Run Project

```
gulp --js
```
```
gulp --sass
```
```
gulp localhost
```

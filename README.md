# Quik Static Pages

## Setup

### Required


```
node >= 16 LTS
```

### Install Gulp
```
cd tools
```

```
npm install --global gulp
```
```
npm install --global gulp-cli
```
```
npm install
```

### Run Project

```
gulp --js
```

```
gulp --sass
```

## compile files in theme/assets folder
```
gulp compile
```

## Run locally to see changes realtime
```
gulp localhost
```
